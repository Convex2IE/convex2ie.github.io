    JSON总结

    what

    1. JSON 指的是 JavaScript 对象表示法(javascript Object Notation)
    2. JSON 是轻量级的文本数据交换格式
    3. JSON 文件的文件类型是".json"
    4. JSON 文本的 MIME 类型是"application/json"

    JSON 名称/值对

    名称/值对包括字段名称（在双引号中），后面写一个冒号，然后是值：

    "name" : "convex"

    JSON 值

    • 数字（整数或浮点数）
    • 字符串（在双引号中）
    • 逻辑值（true 或 false）
    • 数组（在中括号中）
      • 数组可包含多个对象
    • 对象（在大括号中）
      • 对象可以包含多个名称/值对
    • null
