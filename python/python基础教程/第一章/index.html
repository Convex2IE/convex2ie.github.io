    第一章

    1. python是一种解释型语言，其一行一行处理语句

    2. 整除运算//

    3. python x % y = x - ((x // y) * y)

    4. 对于整除运算，重点是它向下圆整。-10 // 3 = -4

    5. 乘方运算的优先级比求负(单目减) 高

    6. 整型数值的表达：

    7. 十六进制：0xAF

    8. 八进制：010
    9. 二进制：0b1011010010

    10. 十进制：123

    11. 使用python变量前必须给它赋值，因为python变量没有默认值

    12. 表达式是一些东西，而语句做一些事情。所有语句的一个根本特征：执行修改操作。

    13. 鉴于函数调用返回一个值，因此它们也是表达式

    14. 可将模块视为扩展，通过将其导入可以扩展python功能：import

    15. 通过module.function的方式使用模块中的函数

    16. 可以通过from module import function导入函数，在调用时无需指定模块前缀

    17. 可以通过变量来引用函数foo = math.sqrt, foo(4) = 2.0 PS：foo指向math.sqrt函数内存地址，跟()后，为执行该段函数代码

    18. 在代码中，#后面到行尾的所有内容都将被忽略

    19. 字符串类型 使用单引号或双引号无差别

    20. 在字符串类型中可以通过转义字符\表示物殊字符

    21. 使用str能以合理的方式将值转换为用户能够看懂的字符串，使用repr通常会获得值的合法python表达式

    22. 长字符串通过三引号‘’‘或“”“表示，其可以通过多行来表示字符串。即无需通过\转义回车

    23. 原始字符串用前缀r表示。 PS：其与函数repr不通之处在于，前者是直接的表达方式，后者是调用函数后的返回值

    24. 字符串的的表达：

      1. 基本表达方式：单引号或双引号括起

      2. 长字符串：三引号括起

      3. 原始字符串：增加前缀r

      4. 可以通过\u， \U, \N{name}的方式指定Unicode字符集

      ```python

      "\u00C6" 'Æ' "\U0001F60A" '😊' "This is a cat: \N{Cat}" 'This is a cat: 🐈' ```

      1. 可以通过使用前缀b，以bytes的方式表达字符串

      ```python

      b'Hello, World!' #可以直接通过ASCII字符表示 b'Hello, World!' b'H\xc3\xa611\xc3\xa5, w\xc3\xb8rld!' #也可以通过 \x + 16进制 表示 b'H\xc3\xa611\xc3\xa5, w\xc3\xb8rld!' b'H\xc3\xa611\xc3\xa5, w\xc3\xb8rld!'.decode("utf-8") #只会使用了decode方法后，才会以字符串的方式表达 'Hæ11å, wørld!' b'Hello, World!'.decode() #默认以utf-8解码 'Hello, World!' b'H\xc3\xa611\xc3\xa5, w\xc3\xb8rld!'.decode() #默认以utf-8解码 'Hæ11å, wørld!' b'H\xc3a611c3a5, w\xc3\xb8rld!'.decode() #因utf-8为非定长的编码，必须用\x的方式分隔每个字符 Traceback (most recent call last): File "", line 1, in UnicodeDecodeError: 'utf-8' codec can't decode byte 0xc3 in position 1: invalid continuation byte ```
