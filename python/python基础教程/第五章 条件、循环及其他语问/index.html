    print 和 import

    打印多个参数

    print 可用于打印一个表达式，这个表达式要么是字符串，要么将自动转换为字符串

    >>> print('Age', 42)
Age 42
>>> 
>>> name = 'Gumby'
>>> salutation = 'Mr.'
>>> greeting = 'Hello,'
>>> print(greeting, salutation, name)
Hello, Mr. Gumby

>>> greeting  = 'Hello'
>>> print(greeting + ',', salutation, name)     #当变量不包含','时，采用 + 运算符相连
Hello, Mr. Gumby

#可自定义分隔符，缺省为空格
>>> print('I', 'wish', 'to', 'register', 'a', 'complaint')
I wish to register a complaint
>>> print('I', 'wish', 'to', 'register', 'a', 'complaint', sep = '_')
I_wish_to_register_a_complaint

#可自定义结束字符串，缺省为换行符
>>> print('Hello,'); print('world!')
Hello,
world!
>>> print('Hello,', end = ''); print('world!')
Hello,world!

>>> import math as foobar

>>> foobar(4)
2.0

    序列解包

    #序列解包(或可迭代对象解包): 将一个序列（或任何可迭代对象）解包，并将得到的值存储到一系列变量中
>>> x, y, z = 1, 2, 3       #注意，右侧是元组
>>> print(x, y, z)
1 2 3
>>> 
>>> x, y = y, x
>>> print(x, y, z)
2 1 3

#要解包的序列包含的元素个数必须与在等号左边列出的目标个数相同，否则python将引发异常
>>> x, y, z = 1, 2
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: not enough values to unpack (expected 3, got 2)
>>> x, y, z = 1, 2, 3, 4
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: too many values to unpack (expected 3)

#可使用星号运算符(*)来收集多余的值，这样无需确保和变量的个数相同
>>> a, b, *rest = [1, 2, 3,4]
>>> rest
[3, 4]
>>> 
>>> name = 'Albus Percival Wulfric Brian Dumbledore'
>>> first, *middle, last = name.split()
>>> middle
['Percival', 'Wulfric', 'Brian']

#注意，带星号的变量最终包含的总是一个列表
>>> a, *b, c = 'abc'
>>> a, b, c
('a', ['b'], 'c')

    代码块

    代码块是一组语句，通过缩进代码（即在前面加空格）来创建。在同一个代码块中，各行代码的缩进量必须相同

    在Python中，使用冒号(:)指出接下来是一个代码块，并将该代码块中的每行代码都缩进相同的程度

    迭代字典

    #遍历字典的关键字
>>> d = {'x': 1, 'y': 2, 'z': 3}
>>> for key in d:
...     print(key, 'corresponds to', d[key])
... 
x corresponds to 1
y corresponds to 2
z corresponds to 3

#for可以使用序列解包
>>> for key, value in d.items():
...     print(key, 'corresponds to', value)
... 
x corresponds to 1
y corresponds to 2
z corresponds to 3
#字典的元素的排列顺序是不确定的。要让映射记住其项的插入顺序，可使用模块collections中的OrderedDict类

    一些迭代工具

    1. 并行迭代

    ```python

    names = ['anne', 'beth', 'george', 'damon'] ages = [12, 45, 32, 102] for i in range(len(names)): ... print(names[i], 'is', ages[i], 'years old') ... anne is 12 years old beth is 45 years old george is 32 years old damon is 102 years old

    #使用内置函数zip实现 names = ['anne', 'beth', 'george', 'damon'] ages = [12, 45, 32, 102] for name, age in zip(names, ages): #注意，for和zip之前的表达式是元组 print(name, 'is', age, 'years old')

    #函数zip可用于“缝合”任意数量的序列，但当长度不同时，将在最短的序列用完后停止“缝合”

    list(zip(range(5), range(100000000))) [(0, 0), (1, 1), (2, 2), (3, 3), (4, 4)] ```

    1. 迭代时获取索引

    python #可能想替换一个字符串列表中所有包含子串'xxx'的字符串 for index, string in enumerate(strings): if 'xxx' in string: strings[index] = '[censored]'

    1. 反向迭代和排序后再迭代

    ```python

    sorted([4, 3, 6, 8, 3]) #sorted返回一个列表 [3, 3, 4, 6, 8] sorted('Hello, world!') [' ', '!', ',', 'H', 'd', 'e', 'l', 'l', 'l', 'o', 'o', 'r', 'w'] reversed('Hello, world!') #reversed返回一个可迭代对象 list(reversed('Hello, world!')) ['!', 'd', 'l', 'r', 'o', 'w', ' ', ',', 'o', 'l', 'l', 'e', 'H'] ''.join(reversed('Hello, world!')) '!dlrow ,olleH' ```

    跳出循环

    1. break

    ```python #要结束(跳出)循环，可使用break

    from math import sqrt for n in range(99, 0, -1): #range可以指定步长 ... root = sqrt(n) ... if root == int(root): ... print(n) ... break ... 81 ```

    1. continue

    ```python #语句continue结束当前迭代，并跳到下一次迭代开头 for x in seq: if condition1: continue if condition2: continue if condition3: continue

    do_something()
do_something_else()
do_another_thing()
etc()

    ```

    1. while True/break成例

    python while True: word = input('Please enter a word: ') if not word: break print('The word was ', word)

    1. 循环中的else子句

    python #循环中的else子句，仅在没有调用break时才执行 from math import sqrt for n in range(99, 81, -1): root = sqrt(n) if root == int(root): print(n) break else: print('Didn't find it!')

    简单推导

    列表推导

    >>> girls = ['alice', 'bernice', 'clarice']
>>> boys = ['chris', 'arnold', 'bob']
>>> [b + '+' + g for b in boys for g in girls if b[0] == g[0]]  #注意，开头是表达式，其结果作为列表的元素
['chris+clarice', 'arnold+alice', 'bob+bernice']


>>> girls = ['alice', 'bernice', 'clarice']
>>> boys = ['chris', 'arnold', 'bob']
>>> letterGirls = {}
>>> for girl in girls:
...     letterGirls.setdefault(girl[0], []).append(girl)    #字典的setdefault方法返回的值实际是字典项的值原本，而非副本
... 
>>> print([b + '+' + g for b in boys for g in letterGirls[b[0]]])
['chris+clarice', 'arnold+alice', 'bob+bernice']

    并没有元组推导，使用()是创建生成器

    字典推导

    >>> squares = {i: "{} squared is {}".format(i, i**2) for i in range(10)}
>>> squares[8]
'8 squared is 64'

    pass, del, exec

    1. pass 什么都不做，适合用作占位符

    2. del 只是删除变量或数据结构的成员，但不能用于删除值。甚至在Python中根本没有办法删除，只有在数值不再使用时，Python解释器会立即将其删除

    3. ```python #函数exec将字符串作为代码执行

      exec('print("Hello, world!")') Hello, world! exec(input('command: ')) command: print('Hello') Hello

    #通过给exec传递第二个参数——字典，用作代码字符串的命名空间

    from math import sqrt scope = {} exec('sqrt = 1', scope) sqrt(4) 2.0 scope['sqrt'] 1 scope.keys() dict_keys(['builtins', 'sqrt'])

    #exec执行一系列Python语句，而eval计算用字符串表示的Python表达式（包括函数）的值，并返回结果

    eval(input("Enter an arithmetic expression: ")) Enter an arithmetic expression: 6 + 18 * 2 42 #为避免eval执行函数影响全局变量，建议也提供单独命名字间 ```
