    第六章 函数

    函数执行特定的操作并返回一个值，可以调用它（调用时可能需要提供一些参数——放在圆括号中的内容）。

    #在def语句后面以及模块和类的开头，添加的字符串称为文档字符串(docstring)
def square(x):
    'Calculates the square of the number x.'
    return x * x

>>> square.__doc__          #__doc__是函数的一个属性
'Calculates the square of the number x.'

>>> help(square)
Help on function square in module __main__:

square(x)
    Calculates the square of the number x.


    #return语句后不跟参数，则返回None值
def test():
    print('This is printed')
    return
    print('This is not')

>>> x = test()      #不要忘记函数也可以执行其它语句，而不仅仅只是return
This is printed
>>> x               #函数的返回值为None，且在执行return语句后结束，第二个print未被执行
>>> print(x)
None

#函数中不包含return语句，则在执行完函数后默认返回None值
>>> def test():
...     print('This is print')
... 
>>> x = test()
This is print
>>> print(x)
None

#注意，如果在if之类的语句中返回值，务必确保其他分支也返回值，以免调用者期望函数返回一个序列时（假设），不小心返回了None

    #在python中，调用函数传递参数，实际上是传递实参的内存地址给形参使用，这点与python中的赋值语句类似。所有要注意，函数内的局部变量的修改可能会影响外部变量。
>>> def change(n):
...     n[0] = 'Mr. Gumby'
...     
... 
>>> names = ['Mrs. Entity', 'Mrs. Thing']
>>> change(names)
>>> names
['Mr. Gumby', 'Mrs. Thing']

    函数可以同时返回多个值，但其实就是一个tuple

    def story(**kwds):
    return 'Once upon a time, there was a ' \
            '{job} called {name}.'.format_map(kwds)


def power(x, y, *other):
    if others:
        print('Received redundant parameters:', others)
    return pow(x, y)


def interval(start, stop = None, step = 1):
    'Imitates range() for step > 0'
    if stop is None:
        start, stop = 0, start
    result = []

    i = start
    while i < stop:
        result.append(i)
        i += step
    return result

    函数的参数有两种：位置参数和关键字参数

    可以结合使用位置参数和关键字参数，但必须先指定所有的位置参数，否则解释器将不知道它们是哪个参数（即不知道参数对应的位置）

    收集参数：定义函数时， 或 * 声明形参为元组或字典

    #分配参数：调用函数时，* 或 ** 将实参从元组、字典翻译成位置或关键字参数
#感受类似于解包
def add(x, y):
    return x + y

>>> params = (1, 2)
>>> add(*params)
3

def hello(greeting = 'Hello', name = 'world'):
    print('{}, {}!'.format(greeting, name))

>>> params = {'name': 'Sir Robin', 'greeting': 'Well met'}
>>> hello(**params)
Well met, Sir Robin!

    #在函数内，使用全局变量
>>> x = 1
>>> def change_global():
...     global x
...     x = x + 1
... 
>>> change_global()
>>> x
2

#即使在执行函数时，在函数中声明的全局变量事先不存在，也能正常工作
>>> def var_global():
...     global y
...     y = 10
... 
>>> var_global()
>>> y
10

    #Python函数在定义的时候，默认参数L的值就被计算出来了，即[]，因为默认参数L也是一个变量，它指向对象[]，每次调用该函数，如果改变了L的内容，则下次调用时，默认参数的内容就变了，不再是函数定义时的[]了。
>>> def add_end(L=[]):
...     L.append('END')
...     return L
... 
>>> add_end()
['END']
>>> add_end()
['END', 'END']
>>> add_end()
['END', 'END', 'END']

    递归函数通常包含下在两部分：

    1. 基线条件（针对最小问题）：满足这种条件时函数将直接返回一个值
    2. 递归条件：包含一个或多个调用，这些调用旨在解决问题的一部分
