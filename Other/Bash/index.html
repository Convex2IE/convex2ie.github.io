    变量设定方式 str没有设定 str为空值 str已设定且不为空值 说明
    var=${str-expr} var=expr var=NULL var=str str/expr
    var=${str:-expr} var=expr var=expr var=str str/expr
    var=${str+expr} var=NULL var=expr var=expr expr/NULL
    var=${str:+expr} var=NULL var=NULL var=expr expr/NULL
    var=${str=expr} str=var=expr str=var=NULL str=var=str 同-增加str修改
    var=${str:=expr} str=var=expr str=var=expr str=var=str 同-增加str修改
    var=${str?expr} expr-->stderr var=NULL var=str expr仅为stderr
    var=${str:?expr} expr-->stderr expr-->stderr var=str expr仅为stderr

    WildCard(Shell常见通配符)
    字符 含义
    * 匹配 0 或多个字符
    ？ 匹配一个字符
    [list] 匹配 list 中的任意单一字符
    [!list] [^list] 匹配除 list 中的任意单一字符
    [c1-c2] 匹配 c1-c2 中的任意单一字符
    {string1,string2,...} 匹配 sring1 或 string2 (或更多)其一字符串
    bash环境中的特殊字符
    符号 内容
    # 注释符号：这个最常被使用在script 当中，视为说明！在后的资料均不执行
    \ 跳脱符号：将"特殊字符或通配符"还原成一般字符
    | 管线(pipe)：分隔两个管线命令的界定
    ; 连续指令下达分隔符号：连续性命令的界定(注意！与管线命令并不相同)
    ~ 使用者的家目录
    $ 取用变量前置字符：亦即是变量之前需要加的变量取代值
    & 工作控制(job control)：将指令变成背景下工作
    ! 逻辑运算意义上的"非" not 的意思！
    / 目录符号：路径分隔的符号
    >, >> 资料流重导向：输出导向，分别是"取代"与"累加"
    <, << 资料流重导向：输入导向
    ' ' 单引号，不具有变量置换的功能($ 变为纯文字)
    " 具有变量置换的功能！($ 可保留相关功能)
    ` ` 两个"` "中间为可以先执行的指令，亦可使用$( )
    ( ) 在中间为子shell 的起始与结束
    { } 在中间为命令区块的组合！
